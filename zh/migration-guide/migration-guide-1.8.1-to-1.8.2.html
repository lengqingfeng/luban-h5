# 鲁班H5 v.1.8.1至 v.1.8.2 迁移指南

升级鲁班H5版本至 v.1.8.2.

主要修改请参见v1.8.2 release log

将 鲁班H5 升级到v.1.8.2 比较简单，只要执行如下命令即可：

# pull 最新代码
# 重新编译预览模块即可
./luban-h5 rebuild_engine

如果有疑问，提 issue 即可
